## LLM Gateway with LiteLLM

### Introduction

LiteLLM is a lightweight LLM gateway that provides a simple and easy-to-use interface for interacting with LLMs. It is a Python library that can be used to interact with LLMs from a variety of providers, including OpenAI, Anthropic, and Google.
